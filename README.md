# 🚀 Rohan Jha – Developer Portfolio Website

Welcome to the repository of my official Developer Portfolio website! This web application serves as a dynamic, interactive showcase of my full-stack engineering skills, technical experience, and core project builds (MERN stack, MVC architectures, and web technologies).

🌐 **Live Website:** [https://rohan-jha-portfolio.vercel.app](https://rohan-jha-portfolio.vercel.app)

---

## 🎨 Design & Key Features

The portfolio is built following modern, premium web design standards, featuring subtle micro-animations, glassmorphic UI elements, and a clean dark/light mode experience:

- **Interactive 3D Project Showcase:** A dynamic 3D circular gallery highlighting my key full-stack builds.
- **Detailed Case Studies:** In-depth technical breakdown of project context, challenges, solutions, and SDLC stages.
- **Sleek UI/UX Animations:** Powered by **GSAP** and **Framer Motion** for premium page transitions, scroll animations, and interactive elements.
- **Fully Responsive Layout:** Optimized for desktop, tablet, and mobile browsers using **Tailwind CSS**.
- **Lenis Smooth Scrolling:** Implemented Lenis smooth scroll wrappers for a premium scrolling feel.
- **Contact Panel:** Custom form setup for direct communications.

---

## 🛠️ Built With

- **Frontend Core:** React, HTML5, CSS3, JavaScript (ES6)
- **Styling:** Tailwind CSS, Material UI (Icons)
- **Animation Frameworks:** GSAP (GreenSock), Framer Motion, AOS (Animate on Scroll)
- **Routing:** React Router DOM (v6 SPA client-side routing)
- **Utilities:** React CountUp, Intersection Observer
- **Deployment:** Vercel (Production Hosting)

## ⚙️ Local Development Setup

To run this portfolio website locally on your system, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RohanJha03/rohan-jha-portfolio.git
   cd rohan-jha-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the site in your browser.

4. **Build for production:**
   ```bash
   npm run build
   ```
